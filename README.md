# 专注结界

专注结界是一款 Windows 专注工具。它可以在一段倒计时内临时封锁 B 站等容易分心的网站，并用抵挡次数、战报和成就反馈帮你守住当前这一段。

## 下载

打开 [Releases](https://github.com/youyukongzhong/focus-jiejie/releases/latest) 下载最新版。

推荐大多数用户下载：

- `FocusJiejie-0.1.5-setup.exe`：安装版，可选择安装位置和是否创建桌面快捷方式。

也可以下载：

- `FocusJiejie-0.1.5-portable.exe`：免安装版，放到固定目录后直接双击运行。

修改 hosts 需要管理员权限。启动结界或恢复 hosts 时，Windows 可能会弹出权限确认，这是正常现象。

## 使用

1. 打开“专注结界”。
2. 选择守界时长和目标结界。
3. 需要时添加额外域名，例如 `https://www.google.com/` 会自动规范为 `www.google.com`。
4. 点击“启动结界”。

倒计时结束后，应用会自动恢复 hosts，并记录本次守界结果。

## 设置

- 可以在右上角“设置”里切换中文/英文。
- 可以开启“开机自启”，应用会在登录 Windows 后启动到托盘。
- 关闭窗口不会直接退出应用，应用会继续在托盘运行。
- 安装版会检查新版本；发现新版后，可以在应用里下载并更新到原安装位置。
- 免安装版如果开启“开机自启”，请不要随意移动 exe 文件位置。

## 功能

- B 站结界预设，覆盖网页、直播、短链和常见相关域名。
- 娱乐网站结界预设，包含 B 站、微博、知乎热榜、抖音网页版。
- 支持 10/25/50/90 分钟预设，也支持 1 到 480 分钟自定义时长。
- 支持附加域名。
- 普通模式可以提前解除。
- 死守模式提前解除需要输入确认文本。
- “恢复 hosts”只用于结界结束后或状态异常时清理封锁状态，守界进行中不能用它绕过提前解除。
- 统计累计分钟、成功局数、抵挡次数、结界石、等级和成就。

## 网站仍能访问怎么办

hosts 封锁对已经打开的浏览器标签页、旧连接或 DNS 缓存不一定立刻生效。

如果启动结界后，已经打开的封锁网站仍能访问，请重启浏览器，或者新开浏览器后再验证。

专注结界不会强制关闭 Chrome/Edge，也不会额外修改防火墙规则，避免影响正常浏览器状态。

## 手动恢复 hosts

正常情况下，应用会自动恢复 hosts。

如果应用异常退出，或者你看到 hosts 状态一直没有恢复，可以在应用里点击“恢复 hosts”。

如果应用内无法恢复，也可以手动打开：

```text
C:\Windows\System32\drivers\etc\hosts
```

删除以下标记之间的内容：

```text
# BEGIN FOCUS-JIEJIE
...
# END FOCUS-JIEJIE
```
